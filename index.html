Everything That's Built

🧠
9-Point Heuristic Engine
IP host, suspicious TLD, phishing patterns, brand impersonation, domain length, hyphens, subdomains, numeric obfuscation, URL path — all scored instantly.
CORE ENGINE
🔒
HTTP Detection
HTTP (non-HTTPS) sites get a +4.0 score penalty. Whitelisted domains get amber badge warning instead of block. Unknown HTTP sites get full scoring.

🛡️
Trusted Whitelist
35+ major domains (Google, Facebook, Amazon, etc.) permanently skip scoring — zero false positives on legit sites.
FALSE-POSITIVE FIX
↩️
Continue Anyway
User can override a block with a 2-step confirmation. Server's /unblock API removes domain from blacklist and navigates to original URL.
NEW v4.0
📊
Live Dashboard
Auto-refreshes every 3s. Shows stat cards, threat gauge, activity timeline (bar chart), live event log table, blacklist panel. Offline overlay if server down.
NEW v4.0
⚡
Multi-threaded Server
Each request spawns its own thread via _beginthreadex. No blocking. Handles concurrent extension + dashboard polling simultaneously.
PERFORMANCE FIX
💾
Persistent Blacklist
blacklist.txt loaded at startup, appended on every new block. Server restart retains all previously blocked domains.
BUG FIX

4 API Endpoints
