How SmartShield Works

Three layers of protection that activate the moment you navigate to any URL — completely silent in the background.

01
🔗
URL Intercepted
The Chrome Extension hooks into the browser's navigation event. Every tab change fires a request to the local C++ engine before the page loads — with zero delay to the user.
CHROME MV3 API
02
🧠
AI Scoring Engine
The C++ server runs 9 heuristic checks in parallel: brand spoofing, suspicious TLD, phishing patterns, domain length, hyphens, subdomains, numeric obfuscation, HTTP detection, and blacklist lookup.
9-POINT ANALYSIS
03
🛡️
Block or Allow
Score below threshold? Browser continues seamlessly. Score above? Tab is redirected to the blocked page instantly. A second deep scan runs in parallel to confirm — eliminating false positives.
DOUBLE VERIFICATION

Installation & Setup

Deploy the SmartShield ecosystem. The engine must be running for the extension to communicate and perform heuristic analysis.

⚙️
1. SmartShield Engine

Compiled C++ WinSock2 backend. Listens for incoming URL verification requests from the browser extension. Auto-starts with Windows.

2. Browser Extension

Chrome MV3 extension. Intercepts tab navigation events and syncs with the local C++ heuristic engine via localhost:8080.

DOWNLOAD EXTENSION (.ZIP)
Version: 4.3 Stable

🛠️ SETUP INSTRUCTIONS

1. Run SmartShield_Engine.exe — server starts on localhost:8080 and auto-hides.
2. Extract SmartShield_Ext.zip, go to Chrome → Extensions → enable Developer Mode → Load Unpacked.
3. The Shield icon will turn cyan with a green ring when the engine is connected.
